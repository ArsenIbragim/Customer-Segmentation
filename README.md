# Customer Segmentation & Churn Prediction

## Описание проекта
Данный проект посвящён анализу клиентского поведения интернет-магазина.
Основные цели проекта:
- сегментация клиентов с помощью RFM-анализа и кластеризации
- выявление ключевых групп клиентов
- прогноз оттока клиентов с использованием машинного обучения

Проект выполнен в формате Jupyter Notebook и ориентирован на задачи Data Science / Machine Learning.

---

##  Dataset
**Источник:** Kaggle — Online Retail Dataset  
Данные содержат информацию о транзакциях:
- идентификатор клиента
- дата покупки
- количество товаров
- цена
- страна

Каждая строка — отдельная покупка.

---

##  Используемые технологии
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

##  Этапы проекта

### 1. Предобработка данных
- удаление пропусков и возвратов
- обработка выбросов
- создание новых признаков

### 2. Exploratory Data Analysis (EDA)
- анализ продаж по времени
- анализ стран и товаров
- анализ распределений

### 3. Feature Engineering
- построение RFM-признаков:
  - Recency
  - Frequency
  - Monetary
- лог-преобразование признаков

### 4. Сегментация клиентов
- масштабирование данных
- подбор числа кластеров (Elbow, Silhouette)
- кластеризация KMeans
- интерпретация сегментов

### 5. Прогноз оттока клиентов
- формирование признаков
- обучение модели Random Forest
- оценка качества (Accuracy, ROC-AUC)
- визуализация результатов

---

##  Результаты
- Выделены ключевые клиентские сегменты (VIP, лояльные, спящие и др.)
- Построена модель прогноза оттока клиентов
- Получены бизнес-рекомендации по удержанию клиентов

---

##  Вывод

от загрузки данных до бизнес-интерпретации результатов.
